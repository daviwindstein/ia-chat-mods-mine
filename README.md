# 🤖 NEON MOD AI - Professional Generator

O NEON MOD AI é uma inteligência artificial avançada capaz de gerar mods complexos em Java para Minecraft.

### 🔥 Funcionalidades:
- **Scripts Java:** Código limpo e profissional para Forge e Fabric.
- **Texturas:** Geração de Resource Packs automáticos.
- **Shaders:** Configurações de iluminação dinâmica.
- **Mundos:** Geração de arquivos `.dat` de mapas customizados.

### 🚀 Como usar:
1. Escolha sua versão (ex: 1.20.1).
2. Selecione o Loader (Forge/Fabric).
3. Descreva sua ideia e clique em gerar!
